# Claim Boundary — HTCE-Origin v1.0

## Allowed claims

HTCE-Origin v1.0 may be described as a clean Q256 toroidal cognitive runtime with bounded simulation-only agency, L1/L2/L3 state, proof-gated memory, protected trace, dialog/action policy, adaptive sleep consolidation and multitask no-regression probes.

The verified scope is the shipped code and generated artifacts.  If a feature has no test, trace or artifact in this package, it is not a v1.0 release claim.

## Prohibited claims

- AGI or general human-level intelligence.
- Consciousness, subjective experience or qualia.
- Biological life.
- Unrestricted real-world autonomy.
- Board-measured hardware energy/performance claims.
- Unbounded open-world understanding.
- Transformer replacement in all tasks.

## architecture freeze

v1.0 freezes the clean-system architecture as one `HTCERuntime` line.  New features must not bypass AIR, proof, policy, evidence, topology or trace gates.
